# Nguyễn Tuấn Việt Portfolio

Single-page React portfolio/CV website for sharing personal information, experience, projects, certificates, skills, and recruitment contact details.

## Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
- Lucide React icons

## Run Locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```

## Content Source

Primary content is extracted from `CV_NguyenTuanViet_DataAnalyst.pdf` and normalized in `src/data/cv.ts`. The original PDF is copied to `public/CV_NguyenTuanViet_DataAnalyst.pdf` for download.
