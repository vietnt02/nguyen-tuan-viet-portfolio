# Project Overview PDR

## Purpose

Build a static React portfolio site for Nguyễn Tuấn Việt to share CV details, project work, and contact information with recruiters.

## Users

- Recruiters reviewing Data Analyst candidates.
- Hiring managers checking experience, projects, certificates, and technical stack.
- Nguyễn Tuấn Việt sharing a concise public profile link.

## Scope

- Single-page portfolio.
- CV download link.
- Sections for hero, experience, education, activities, projects, certificates, skills, and contact.
- No backend, authentication, CMS, or analytics in this version.

## Success Criteria

- Resume content visible and readable on mobile and desktop.
- Contact actions work for email, phone, LinkedIn, and location.
- PDF download works.
- `npm run lint` and `npm run build` pass.
